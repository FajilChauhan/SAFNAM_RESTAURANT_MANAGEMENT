# SAFNAM Restaurant Backend

Production-ready backend foundation for a Restaurant ERP, POS, and Hotel Management System.

## Completed

- Express application bootstrap
- TypeScript configuration
- Environment validation
- PostgreSQL connection lifecycle through Prisma
- Global API response and error helpers
- Async handler utility
- Centralized error, not found, and request logger middleware
- Helmet, CORS, compression, Morgan, cookie parser, JSON parser, and URL encoded parser
- Graceful shutdown
- `GET /health`
- Placeholder module folders only

## Intentionally Deferred

- Authentication
- Business APIs
- Prisma models
- Database migrations
- Redis
- Socket.IO runtime setup
